## Spacing

Pick from 4px grid: 4, 8, 12, 16, 20, 24, 32, 48, 64, 96. Inside group < between groups < between sections. Padding ≥ gap in same container. Vertical padding > horizontal at equal values (compensate: py={10} px={20}). Once picked, stay consistent for same element type.

Spacing hierarchy makes the design feel expensive:

- Canvas margin: protects the whole composition and keeps key text away from edges.
- Section gap: the largest recurring space; separates ideas.
- Group gap: separates clusters inside one section.
- Component gap: separates card/icon/text elements.
- Micro spacing: icon-to-label, eyebrow-to-title, title-to-body.

Whitespace is a design element, not leftover space. Do not fill the canvas just because space remains. Text must not touch card edges, icons must not crowd labels, and dense content needs extra breathing room around it.

Spacing must create rhythm, not equal padding everywhere. Use smaller steps for micro relationships, medium steps for cards/groups, and larger steps for section separation or editorial breathing room.

Always check breathing room, safe margins, edge pressure, and visual density. If a design feels noisy, first increase grouping clarity and remove elements before adding more decoration.
